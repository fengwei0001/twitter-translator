Twitter 翻译扩展隐私政策

引言

欢迎使用 Twitter 翻译扩展。我们非常重视您的隐私权，并致力于保护您的个人信息。本隐私政策旨在说明我们如何收集、使用和保护您的信息。

信息收集

我们收集的信息

在使用本扩展时，我们仅收集以下必要信息：

  • 您在 Twitter/X 平台上浏览的公开推文内容
  • 推文作者的公开显示名称

我们不收集的信息

我们不会收集或存储：

  • 您的个人身份信息
  • 您的 Twitter/X 账号信息
  • 您的浏览历史
  • 您的设备信息

信息使用

我们收集的信息仅用于：

  • 提供推文翻译服务
  • 改进扩展功能和用户体验

所有收集的推文内容仅在翻译过程中临时使用，不会进行永久存储。

第三方服务

API 服务

我们使用 Deepseek API 提供翻译服务。在使用过程中：

  • 推文内容会被发送至 Deepseek 服务器进行翻译
  • 所有数据传输均采用安全加密方式
  • 翻译完成后不会在服务器保留原文或译文

数据安全

我们采取以下措施保护您的信息：

  • 使用 HTTPS 加密所有数据传输
  • 仅在必要时临时处理数据
  • 不进行任何形式的数据存储

用户权利

作为用户，您拥有以下权利：

  • 随时停止使用本扩展
  • 通过卸载扩展完全移除所有相关数据
  • 查看本隐私政策的更新内容

政策更新

我们可能会不时更新本隐私政策。更新后的政策将在扩展更新时生效，您可以随时在扩展文档中查看最新版本。
